# Requesting new software

## Pre-approved tools (no ticket needed)

Acme has standing licenses for these — just install and go:

- Slack, Linear, Notion
- GitHub Desktop, VS Code, JetBrains IDEs
- Figma, Zoom
- 1Password, Tailscale (VPN)

Install from the company app catalog or the vendor's site. License
auto-assigns from your Okta groups.

## Everything else: file a Linear ticket

In the **IT** project, include:

- **Name** of the tool
- **Vendor** + URL
- **Monthly or per-seat cost**
- **Business justification** (one paragraph)
- **Data handling**: does this tool process customer data? If yes, IT will
  loop in security review BEFORE purchase

## Security-reviewed list

A separate Notion page tracks tools that have already passed security
review. Ask IT for the current list before re-requesting something that
was previously denied.

## Expected turnaround

- Pre-approved → instant
- Standard ticket → 2 business days
- Security review required → 5-7 business days

## Bot etiquette

When an employee asks "can I use X?", the bot should:
1. Check the pre-approved list first
2. If not pre-approved, file the ticket on their behalf using their
   answers to the required fields (don't make them re-type)
3. Set expectations on turnaround based on whether security review is needed
