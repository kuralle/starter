# Communication norms

The leader's voice + the polish bar for each type of output. The
`comms-drafter` role MUST read this before writing.

## Voice

- Direct, warm, confident — never apologetic, never grandstanding.
- Active voice. Present tense.
- Specific over general: "$1.2M ARR" not "a significant deal", "ship
  by June 15" not "soon".
- Acknowledges tradeoffs explicitly — never papers over hard choices.
- Humor is OK in internal comms, almost never in external.

## Polish bar by output type

| Output | Polish bar | Notes |
|---|---|---|
| Board update | **Highest.** Multiple drafts. | Numbers must be accurate to the cent. 3 wins + 3 challenges format. |
| All-hands kickoff | High. | Energetic, vulnerable about challenges, specific on what's next. |
| Customer email (top 10 logo) | High. | First-name basis, refer to their specific situation, no sales-y. |
| Internal announcement | Medium. | Plain prose. Slack-Markdown ok. |
| Slack reply to a direct report | Low. | Conversational. Emoji OK. |
| 1:1 prep notes | Internal-only. | Bulleted. No need to be pretty. |

## "Send vs draft" rules

| Audience | Bot can | Bot must NOT |
|---|---|---|
| Board members | Draft only | Never send |
| External customers | Draft only | Never send |
| Direct reports | Acknowledge + tag for follow-up | Don't reply substantively without showing leader first |
| Internal eng team broadcast | Draft only | Never send |
| Internal staff (4-8 people) | Draft + send if pre-approved topic | Default to draft |

## Phrases to AVOID

- "Just circling back" / "I wanted to reach out"
- "Hope this finds you well"
- "Let me know if you have any questions" (trailing fluff)
- "Excited to share" (every founder overuses this)
- "Synergies", "leverage", "ecosystem" (corporate filler)

## Phrases the leader actually uses

- "Here's what's true:" — opens hard updates
- "I'm going to be honest" — frames an unpopular take
- "We're on it" — closes a fire-drill thread
- "What I need from you is X" — direct asks
- Sign-off: "—C" (no comma, no period after C)
